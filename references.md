# 资料校准与来源记录

本文用于记录本项目第一版知识图谱的参考底盘。资料使用目标是校准核心概念完整性，不进行大段引用。

## 主要来源

### OpenStax, Principles of Management

- 链接：https://openstax.org/details/books/principles-management
- 用途：核对本科“管理学原理”课程的主流范围，包括计划、组织、领导、控制、战略、人力资源、激励、沟通、团队、组织结构、变革等模块。
- 使用方式：作为章节范围和概念底盘参考，未直接照搬目录。

### OpenStax, Principles of Management Index

- 链接：https://openstax.org/books/principles-management/pages/index
- 用途：核对关键术语覆盖情况，如管理者角色、组织环境、PESTEL、战略、组织结构、激励理论、沟通、团队、控制、变革与创新。
- 使用方式：用于查漏补缺，确认图谱核心节点没有明显遗漏。

### XMind 官方 Markdown 导入说明

- 链接：https://xmind.com/help/import-markdown-file-web
- 用途：确认 XMind 支持导入 Markdown 文件，并可通过上传文件或粘贴 Markdown 文本生成思维导图。
- 使用方式：指导 `xmind-import.md` 采用标题层级优先、短列表补充的格式。

### XMind 官方导入格式说明

- 链接：https://xmind.com/user-guide/import-new
- 用途：确认 XMind 支持 Markdown、OPML、Word 等多种导入格式。
- 使用方式：支撑本项目选择 Markdown 作为 AI 阶段主源稿。
